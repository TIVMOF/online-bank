## Application
## Build

	docker build -t codbex-petstore .
 
## PULL

	docker pull ghcr.io/codbex/codbex-petstore:latest

## Run

	docker run --name codbex-petstore -d -p 8080:8080 ghcr.io/codbex/codbex-petstore:latest
 
## Clean

	docker rm codbex-petstore
